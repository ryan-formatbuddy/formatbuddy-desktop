# 자주 묻는 질문 (FAQ)

## 설치 / 실행

### 설치하려고 하니 "Windows의 PC 보호" 경고가 떠요
정상이에요. 포맷버디는 아직 **코드 사이닝 인증서가 없는 unsigned 인스톨러**라서 Windows SmartScreen이 한 번 막아섭니다.

**진행 방법:**
1. 경고 창에서 `추가 정보 (More info)` 클릭
2. 그러면 아래에 `실행 (Run anyway)` 버튼이 나타나요
3. 클릭하면 설치 진행됩니다

이 경고는 "포맷버디가 위험하다"는 뜻이 아니라 "Windows가 이 앱의 제작자를 아직 모른다"는 뜻이에요. 다운로드 수가 쌓이거나 인증서를 도입하면 자동으로 사라집니다.

### 회사/학교 PC에서 설치가 막혀요
IT 정책으로 unsigned 앱이 차단되어 있을 가능성이 높습니다. 관리자에게 예외 처리를 요청하시거나, **개인 PC에서 사용**해 주세요.

### 어떤 Windows에서 동작하나요?
- Windows 10 (1903 이상 권장)
- Windows 11
- x64 (64비트) 전용. ARM Windows는 아직 미지원.

## 진단

### 진단이 얼마나 걸리나요?
- Quick scan: 평균 1~3분 (PC 환경에 따라 다름)
- 백업 manifest: 사용자 폴더 크기에 따라 수십 초 ~ 수십 분 (Downloads 폴더가 크면 오래 걸려요)

### 무엇을 수집하나요?
**모두 Ryan의 PC 안에서만 동작합니다. 서버로 어떤 파일도 보내지 않아요.** 자세한 건 [개인정보처리방침](PRIVACY_POLICY.md) 참고.

수집하지 **않는** 것:
- 공동인증서 개인키 / 비밀번호
- 브라우저에 저장된 비밀번호
- Wi-Fi 비밀번호 (프로필 *이름*만)
- 카카오톡 메시지 내용
- 그 외 모든 비밀번호 / 토큰 / 인증 정보

수집하는 것 (모두 로컬 JSON 파일에만):
- 시스템 정보 (모델, OS, CPU, 메모리)
- 디스크 용량
- 사용자 폴더 크기 (Desktop/Documents/Downloads 등)
- 설치된 앱 이름 / 버전 / 게시자
- 드라이버 목록
- Wi-Fi 프로필 *이름*
- 공동인증서가 어디 *있는지* (경로만, 내용 X)
- winget으로 다시 설치 가능한 앱 목록 (옵션)

### 백업 manifest는 뭐예요?
사용자 폴더(Desktop/Documents/Pictures/Music/Videos/Downloads)에 있는 모든 파일의 **SHA-256 해시 + 크기 + 수정 시각**을 JSON으로 정리한 거예요.

포맷 후 백업에서 복원할 때, manifest와 비교하면 **모든 파일이 똑같이 돌아왔는지 검증**할 수 있어요. (수동 검증 도구는 별도 또는 PowerShell로 직접 — 가이드는 추후 추가 예정)

기본 설정으로 **100MB 이하 파일만** 해시합니다 (큰 ISO/동영상은 skip — 너무 오래 걸려서). 추후 옵션화 예정.

### winget이 뭐예요? 왜 표시되나요?
Windows 10/11에 내장된 패키지 관리자예요 (`winget install Google.Chrome` 같은 명령). 포맷버디는 설치된 앱 중 winget으로 다시 설치 가능한 것의 목록을 JSON에 저장해서, 포맷 후 한 줄 명령으로 앱 재설치를 도와줍니다.

PC에 winget이 없으면 (오래된 Windows 10 등) 그냥 해당 섹션이 비어있어요.

## 사용 후 / 복원

### JSON 리포트는 어디에 저장돼요?
"JSON으로 저장" 버튼을 누르면 **저장 위치를 직접 선택**할 수 있어요. 기본 경로는 데스크탑.

웹 리포트 뷰어(`formatbuddy.vercel.app/report/import`)에서 그 JSON을 업로드하면 보기 좋게 정리됩니다.

### 포맷 후에 어떻게 써요?
1. 포맷 전: 포맷버디 실행 → 진단 → JSON 저장 (USB나 클라우드에 백업)
2. 포맷 후: 새 Windows에서 다시 다운로드 → JSON import → 체크리스트 따라 복원

### 자동으로 백업해주나요?
**아니요.** 포맷버디는 **백업 도구가 아니라 백업 동행 도구**예요. 무엇을 백업해야 하는지 알려주고, 백업 후 manifest로 검증을 도와주는 것까지. 실제 백업은 OneDrive / Google Drive / 외장하드 등 Ryan이 익숙한 도구로 하세요.

## 업데이트

### 새 버전은 어떻게 받아요?
**v0.2.0 이상**이면 앱이 자동으로 체크해서 받아요. 새 버전이 다운로드되면 화면 아래에 안내 배너가 뜹니다:

> 준비 끝났어요. 다시 켜면 새 버전으로 시작해요  [지금 재시작]

[지금 재시작] 누르면 새 버전으로 자동 재실행됩니다.

### v0.1.x 사용 중인데 자동 업데이트가 안 돼요
v0.1.x는 자동 업데이트 기능이 없는 버전이라 한 번 수동으로 v0.2.0 이상을 받아주셔야 해요. 그 다음부터는 자동.
