# FormatBuddy 톤 가이드 (v2.0)

이 문서는 사용자가 보는 한국어 카피를 작성·수정할 때 따르는 규칙이에요.
모든 카피는 [src/shared/copy.ts](../src/shared/copy.ts)에 모여 있고, 추가로
[tests/ui-tone-guard.test.ts](../tests/ui-tone-guard.test.ts)가 금지어를 빌드
시점에 막아줘요. 이 문서에 새 단어를 추가했으면 그 테스트의 BANNED_PHRASES도
함께 업데이트해주세요.

---

## 큰 원칙

1. **사용자 PC를 지키는 게 우선.** 표현이 자신을 부풀리는지, 실제로
   한 일 이상을 약속하는지 항상 살펴요.
2. **"포맷버디"가 한 일과 "Windows"가 한 일을 분리.** 백신·시스템 복원·휴지통
   비우기는 모두 Windows가 처리해요. 우리는 보여주고 안내할 뿐.
3. **반말 금지, "Ryan" 이름은 절대 카피에 박아넣지 않음.** 코드 변수명에는
   사용자명이 들어가지만, 사용자에게 보이는 텍스트는 "이 PC" "사용자"로.
4. **"~할게요" "~보여드릴게요" "함께 봐요"** 같은 옆에 앉아 도와주는 톤 유지.

---

## 권장 어휘

| 권장 | 회피 | 메모 |
|------|------|------|
| 살펴볼게요 / 살펴봤어요 | 스캔 / 진단 완료 | 진행 중일 때 "살펴보는 중" |
| 정리 / 지우기 | 삭제 / 제거 | 사용자 입장에서 "지우기"가 더 친숙 |
| 앱 | 프로그램 | 단, Windows UI 라벨 인용 시 "프로그램" OK |
| 챙길게요 | 처리할게요 | 사용자 데이터를 다룰 때 |
| 같이 봐요 | 확인하세요 | 안내 톤 유지 |
| 한 번 더 살펴보세요 | 반드시 확인하세요 | 강요 톤 회피 |
| 시작 앱 | 시작 프로그램 | "앱" 통일 원칙 |
| 큰 파일 | 대용량 파일 | 부드러운 표현 |
| 휴지통으로 보냈어요 | 삭제했어요 | 되돌릴 수 있음을 암시 |

## 금지 표현 (테스트로 막혀 있음)

| 금지어 | 이유 | 대안 |
|--------|------|------|
| 치료 완료 / 감염 발견 / 바이러스 제거 / 악성코드 제거 | 백신처럼 동작하지 않음 | "Windows 보안에서 확인했어요" |
| 스캔 완료 | 차가운 톤 | "살펴봤어요" |
| 심각한 상태 | 과장 | "지쳐 있어요" / "포맷이 가장 깔끔해요" |
| PC를 리셋 / 초기화 | 행동 약속 | "포맷 동행" |
| 자동 삭제 / 자동 처리 (긍정 컨텍스트) | 자동 처리 안 함 | "Ryan이 직접 선택해요" |
| 100점 달성 / 만점 확보 | 가짜 품질 착시 | 점수만 표시, 단어 없이 |
| 필수입니다 | 강요 | "권장해요" / "한 번 보면 좋아요" |

부정 컨텍스트("자동 삭제하지 않아요" 같은 안전 가드 표현)는 허용. 테스트의
`unless` 정규식으로 처리.

## 정보 밀도

- **카드 한 장에 한 가지 결정.** 카드 안에 여러 액션이 섞이면 사용자가
  무엇을 골라야 할지 헷갈려요. CTA는 한 카드에 한 개 + 보조 액션은 작은 링크.
- **숫자는 단위와 함께.** "12.3 GB", "5개 항목", "14일 전" 식. 단위 없는
  숫자는 디자이너 의도가 분명할 때만.
- **시간 표현은 절대 시각이 아니라 상대 시각.** "2시간 전 점검" "3일 전".
- **불확실은 솔직히.** 정확히 측정 못 한 항목은 "확인 필요" 표기.

## 액션 카피 패턴

CTA 버튼은 동사 + 명사 패턴.

| 좋음 | 회피 |
|------|------|
| 안전 정리 시작 | 정리하기 |
| 보안 점검 열기 | 보안 |
| 다시 점검 | 새로고침 |
| 처음으로 | 닫기 |
| Windows 휴지통 열기 (되돌리기) | 휴지통 |

괄호 안 설명은 사용자가 행동을 후회할 수 있는 비가역 상황(휴지통 비우기,
영구 삭제 등)에서만 사용.

---

## 위험 안내

위험이 있는 작업 전에는 **사용자에게 무엇이 일어날지 + 무엇이 되돌려질지**
같이 알려요.

```
"휴지통으로 N개 항목을 보내요. 마음이 바뀌면 휴지통에서 복원할 수 있어요."

"영구 삭제는 되돌릴 수 없어요. 정말 확신할 때만 사용해주세요."

"앱 제거는 Windows 기본 제거 마법사로 진행돼요. 잔여 폴더는 마법사가 끝난 뒤
별도로 정리할 수 있어요."

"포맷버디가 위협을 직접 치료하지 않아요. Windows Defender에서 빠른 검사를
시작하고 결과는 Windows 보안에서 확인해주세요."
```

## i18n 준비

v2.0은 한국어 단일. 그러나 다음 원칙은 미리 지킴:

- 카피는 모두 `copy.ts`에 모음 (component 안에 한국어 문자열 박지 않음)
- 동적 문자열은 함수로: `summary: (count: number) => string`
- 영어 단어를 본문에 섞을 때 한 칸 띄움: "Windows 보안", "Defender 빠른 검사"
