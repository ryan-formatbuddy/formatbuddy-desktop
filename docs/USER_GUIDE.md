# 포맷버디 사용 가이드

> PC 포맷, 처음이거나 오랜만이라 막막하시죠? 포맷버디가 옆에서 같이 챙겨드릴게요.

## 0. 시작하기 전에

이 가이드는 **Windows 10/11 사용자**를 대상으로 합니다. 포맷버디는 PC를 자동으로 포맷하지 않아요. **무엇을 챙겨야 하는지 알려주고, 빠뜨린 게 없는지 같이 살펴주는 동행 도구**입니다.

---

## 1. 다운로드 + 설치

### 1-1. 다운로드

최신 버전: https://github.com/ryan-formatbuddy/formatbuddy-desktop/releases/latest

`FormatBuddy-Setup-X.Y.Z-x64.exe` 파일을 받으세요. (현재 약 85 MB)

> _📸 스크린샷 자리 — Releases 페이지 다운로드 위치_

### 1-2. SmartScreen 경고가 떠요

설치 시 아래 같은 경고가 나올 수 있어요:

```
Windows의 PC 보호
Microsoft Defender SmartScreen이 알 수 없는 앱의 시작을 차단했습니다.
```

**정상이에요.** 자세한 건 [FAQ — SmartScreen](FAQ.md#설치하려고-하니-windows의-pc-보호-경고가-떠요) 참고.

해결: `추가 정보` → `실행` 클릭.

> _📸 스크린샷 자리 — SmartScreen 경고 + 실행 버튼_

### 1-3. 설치 마법사

기본값 그대로 진행해도 됩니다. 설치 경로를 바꾸고 싶으면 `다른 폴더` 선택.

설치가 끝나면 데스크탑에 포맷버디 아이콘이 생겨요.

> _📸 스크린샷 자리 — 인스톨러 첫 화면_

## 4. 리포트 보기

진단이 끝나면 결과 화면으로 자동 전환됩니다.

### 4-1. 상단 카드 (시스템 요약)

- **이 PC**: 모델 / OS / CPU / 메모리
- **저장 공간**: 총 용량 / 여유 공간 / 드라이브별
- **같이 챙길 것**: 공동인증서 후보 / 클라우드 / Wi-Fi 프로필 / BitLocker
- **설치된 앱 / 드라이버**: 개수 / winget 사용 가능 여부 / 프린터
- **사용자 폴더**: Desktop, Documents, Downloads 등의 크기
- **winget으로 다시 설치할 앱**: 포맷 후 한 줄 명령으로 재설치 가능한 앱 수

> _📸 스크린샷 자리 — Report 카드 그리드_

### 4-2. 포맷 전 체크리스트

화면 중앙에 정리됩니다. 하나씩 확인하면서 백업하세요.

### 4-3. JSON으로 저장

`JSON으로 저장` 버튼 클릭 → 저장 위치 선택 (USB나 클라우드 권장).

이 JSON은 포맷 후에도 사용해요. **꼭 PC 밖에 저장**해주세요.

> _📸 스크린샷 자리 — Save Dialog_

### 4-4. 웹 리포트 뷰어 열기

`웹 리포트 뷰어 열기` 클릭 → 브라우저에서 `formatbuddy.vercel.app/report/import` 열림.

거기에 방금 저장한 JSON을 업로드하면 더 보기 좋게 정리됩니다 (스마트폰에서도 열어볼 수 있어요).

## 5. 빠진 파일 확인 목록 (선택)

Report 화면 하단에 파란 박스가 있어요:

```
빠진 파일 확인 목록

Desktop·Documents·Downloads 같은 사용자 폴더의 파일을
해시(SHA-256)로 정리해 저장해요. 포맷 후 복원할 때
같은 파일이 잘 돌아왔는지 확인할 수 있어요.

[빠진 파일 확인 목록 만들기]
```

> _📸 스크린샷 자리 — 빠진 파일 확인 목록 섹션_

`빠진 파일 확인 목록 만들기` 클릭 → 저장 위치 선택 → 진행.

**시간**: 사용자 폴더 크기에 따라 수십 초 ~ 수십 분. Downloads에 큰 파일(영화, ISO 등)이 많으면 오래 걸려요.

**결과**: 모든 파일의 (경로, 크기, SHA-256 해시, 수정 시각)이 들어있는 큰 JSON.

**용도**: 포맷 + 복원 후, 이 JSON과 복원된 파일을 비교해서 **빠뜨리거나 손상된 파일을 찾아낼 수 있어요.** (전용 검증 도구는 별도 — 추후 추가 예정)

## 6. 포맷 진행 (직접)

포맷버디는 포맷 자체는 안 합니다 (안전상). 사용자가 직접:

- Windows 설정 → 시스템 → 복구 → 이 PC 초기화 → 모두 제거
- 또는 USB 부팅 + Windows 설치 미디어

포맷 전 마지막 확인:
- [ ] JSON 리포트가 PC 밖에 저장돼 있나요?
- [ ] 빠진 파일 확인 목록이 있나요? (선택)
- [ ] 사용자 폴더가 모두 백업됐나요?
- [ ] 공동인증서를 옮기셨나요? (NPKI 폴더 통째로)
- [ ] Wi-Fi 비밀번호를 알고 계시나요? (포맷버디는 이름만 알려드려요)
- [ ] 카카오톡 백업 코드를 받아두셨나요? (카카오톡 설정 → 채팅 → 대화 백업)

---

## 7. 복원

포맷 + Windows 재설치 후:

1. 새 Windows에서 포맷버디 다시 다운로드 + 설치
2. 저장해둔 JSON 리포트 import (`/report/import`)
3. 체크리스트에 따라 한 줄씩:
   - winget 명령으로 앱 재설치
   - 백업에서 사용자 폴더 복원
   - 공동인증서 복원
   - 클라우드 동기화 다시 연결
