# 개인정보 처리방침 (초안)

> **⚠️ 법무 검토 면책**: 이 문서는 일반적인 한국 개인정보 보호법(PIPA) 원칙을 따른 **초안**입니다. 실제 출시 전에 변호사 또는 한국 개인정보 보호위원회(PIPC) 가이드라인을 통한 법무 검토를 받아주세요.

---

## 1. 서비스 개요

**포맷버디 데스크탑 (FormatBuddy Desktop)** (이하 "본 서비스")은 Windows PC 포맷 전 챙겨야 할 항목을 점검해주는 **100% 로컬 동작** 데스크탑 애플리케이션입니다.

본 서비스는 **사용자의 개인정보를 수집·저장·전송하지 않습니다.** 모든 진단 정보는 사용자의 PC 안에서만 처리되며, 사용자가 명시적으로 export한 JSON 파일만 사용자의 통제 하에 저장됩니다.

## 2. 수집하지 않는 정보

본 서비스는 다음 정보를 **절대 수집하지 않습니다**:

- 공동인증서(NPKI) 개인키
- 공동인증서 / 인증서 비밀번호
- 브라우저(Chrome, Edge, Firefox, Whale 등) 저장 비밀번호
- Wi-Fi 비밀번호 — 진단에서는 프로필 *이름*만 수집해요. 사용자가 별도
  "Wi-Fi 프로필 백업" 화면에서 **"비밀번호 평문 포함 (위험)" 체크박스를
  명시적으로 켜야만** Windows의 Wi-Fi 내보내기 기능으로 만든 XML 파일에 평문 비밀번호가 포함되며,
  그 XML 파일은 사용자가 직접 고른 폴더에만 저장돼요. 본 서비스는 그
  파일을 어디로도 전송하지 않아요.
- 카카오톡 / 메신저 메시지 내용
- 이메일 / 클라우드 / 소셜 계정 자격 증명
- 카드 번호 / 은행 정보 / 결제 정보
- 그 외 일체의 비밀번호 · 토큰 · 인증 정보

## 2a. v2에서 새로 추가된 로컬 파일 (외부 전송 없음)

본 서비스가 사용자 PC의 `%APPDATA%/FormatBuddy/` 경로에 저장하는 로컬 파일:

- `formatbuddy-state.json` — 진단 점검 기록 (점수, 추천 수치 등) + 정리
  후보 무시 목록. 진단 데이터는 사용자가 직접 export하지 않는 한 외부로
  나가지 않아요.
- `formatbuddy-monitor-prefs.json` — 트레이 / PC 시작 시 켜기 / 주기 알림 /
  정기 자동 점검 예약 / 업데이트 채널 / 복원 지점 자동 생성 / 화면 모드 /
  익명 사용 통계 허용 여부 같은 사용자 설정.
- `formatbuddy-cleanup-log.json` — 정리 실행 내역 (어떤 카테고리에서 몇 개
  항목을 포맷버디 복구함으로 보냈는지). 파일 경로는 저장하지 않아요.
- `formatbuddy-trash/` — 사용자가 직접 고른 정리 항목을 30일 동안 보관하는
  포맷버디 복구함 폴더. 되돌리기를 위해 원래 경로와 만료 시각을 로컬
  복구 기록 파일에 저장하며, 30일이 지난 항목은 앱이 자동으로 비울 수
  있어요.
- `formatbuddy-registry-backups/` — 앱 삭제 후 남은 실행 경로, 앱 연결,
  우클릭 메뉴 같은 흔적을 사용자가 직접 고른 경우, 되돌리기를 위해 필요한 백업을 30일 동안 보관하는 로컬 폴더.
  앱 이름, 백업 파일, 만료 시각만 PC 안에 저장해요.
- `formatbuddy-startup-disabled/` — 사용자가 직접 고른 시작 항목을 잠시
  꺼둘 때, 원래 위치로 되돌릴 수 있도록 30일 동안 보관하는 로컬 폴더.
  30일이 지난 항목은 앱이 자동으로 비울 수 있어요.
- `formatbuddy-audit-log.json` — 통합 감사 로그 (정리 / 앱 제거 /
  Windows 보안 검사 / 설정 변경). 90일 후 자동 정리. 사용자가 활동 기록
  화면에서 직접 열람할 수 있어요. 30일 안에 되돌릴 수 있는 항목 수와
  보관 항목 ID를 남길 수 있지만, 비밀번호나 인증 정보는 저장하지 않아요.

사용자가 직접 export하는 파일 (사용자가 고른 경로):

- 드라이버 백업 폴더 (Windows 드라이버 내보내기 결과)
- Wi-Fi 프로필 XML (Windows Wi-Fi 내보내기 결과)
- 진단 리포트 (HTML / JSON)
- 빠진 파일 확인 목록 (JSON)

본 서비스는 이 export 파일들을 어디로도 전송하지 않아요.

## 3. PC 안에서 처리하는 정보 (서버 전송 X)

본 서비스는 PC 진단을 위해 다음 정보를 **PC 내부에서만 처리**합니다. 서버로 전송하지 않습니다.

### 3-1. 시스템 정보
- 제조사 / 모델
- 시리얼 번호 (마지막 4자리만 표시, 나머지 마스킹)
- OS 종류 / 버전
- CPU / 메모리
- GPU 목록
- 디스크 용량 / 여유 공간

### 3-2. 사용자 폴더 정보
- Desktop / Documents / Pictures / Music / Videos / Downloads 폴더의 총 용량
- (빠진 파일 확인 목록 선택 시) 각 폴더의 파일 목록 + 크기 + SHA-256 해시 + 수정 시각

### 3-3. 소프트웨어 정보
- 설치된 앱 이름 / 버전 / 게시자
- 드라이버 목록 (DeviceName, Manufacturer, DriverDate)
- 프린터 목록
- 브라우저 설치 여부 (실행파일 존재 여부만)
- 앱 다시 설치 준비 목록 (가능한 경우)

### 3-4. 네트워크 / 보안 정보
- Wi-Fi 프로필 *이름* (비밀번호 X)
- 공동인증서 폴더 후보 *경로*의 존재 여부 (내용 X)
- 클라우드 동기화 폴더(OneDrive, Google Drive, Dropbox) *경로*의 존재 여부
- BitLocker 볼륨 암호화 상태

## 4. 정보 저장 위치

- **임시 보관**: 진단 진행 중 `%TEMP%\formatbuddy-scans\` 디렉토리에 임시 JSON 저장. 진단 완료 후 즉시 삭제.
- **사용자 export**: 사용자가 `JSON으로 저장` 또는 `빠진 파일 확인 목록 만들기` 버튼을 명시적으로 누른 경우에만 사용자가 선택한 경로에 저장. 이 파일의 관리는 전적으로 사용자 책임.

## 5. 정보의 외부 전송

본 서비스는 **사용자 PC 외부로 어떤 데이터도 전송하지 않습니다**, 단 다음 두 가지 예외:

1. **자동 업데이트**: 새 버전 확인을 위해 GitHub Releases API(https://api.github.com)에 anonymous 요청을 보냅니다. 사용자 식별 정보는 포함되지 않으며, GitHub의 개인정보 처리방침 (https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)이 적용됩니다.
2. **사용자 명시 동의**: 사용자가 `웹 리포트 뷰어 열기` 버튼을 누르면 기본 브라우저로 https://formatbuddy.vercel.app/report/import 페이지가 열립니다. 그 페이지에 진단 JSON을 업로드할지는 사용자가 직접 선택합니다.

익명 사용 통계는 현재 전송 기능이 연결되어 있지 않습니다. 설정값은 향후 opt-in
기능을 위한 로컬 동의 상태일 뿐이며, 전송 기능을 실제로 추가할 때는 전송 항목과
목적을 이 문서에 다시 명시해야 합니다.

## 6. 사용자의 권리

본 서비스는 사용자 정보를 수집/저장하지 않으므로, 통상의 개인정보 관련 권리(열람, 정정, 삭제, 처리 정지 등) 행사 대상이 없습니다. 사용자가 export한 JSON 파일의 관리·삭제는 전적으로 사용자 본인의 권한입니다.

## 7. 만 14세 미만 아동

본 서비스는 아동의 개인정보를 수집하지 않습니다. 다만 부모/법정대리인의 감독 하에 사용하시기를 권장합니다.

## 8. 보안

- 본 서비스는 앱에 포함된 Windows 점검 파일을 실행하기 전에 **SHA-256 무결성 검증**을 거칩니다. 변조된 점검 파일은 실행이 거부됩니다.
- 외부 점검 파일이 아닌 앱 번들에 포함된 검증된 점검 파일만 실행됩니다.

## 9. 변경 사항

본 처리방침이 변경될 경우 GitHub 저장소의 commit history에서 변경 이력을 확인할 수 있습니다.
